### 已实现的功能模块
- CPU、PPU、APU、内存
- 手柄输入控制(P1、P2)
- 6502汇编调试器(反汇编)，名称表、图案表、精灵等可视化以及内存的dump
- 游戏的加速模式以及存档与读档
- 基于xBRZ算法的图像增强
- 金手指的支持
- 基于UDP协议的网络联机(包含创建、加入、退出、销毁游戏房间的操作)
